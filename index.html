Network Graph Visualization

This tool allows you to visualize relationships between words in a network graph format, providing insights into data patterns. Click the Button below to download data that can be used for this website. This website works when using data from https://constellate.org/ when the full text is requested

Upload JSONL File

Select a JSONL file to upload. This file should contain the data for visualization. The Current file only shows One journal; 'Transactions of the Historical Society of Ghana". Download the file with the "download data" button to analyse 5 more much larger historical journals.

Select Journal

Choose a journal from the list. This will filter the data specific to the selected journal.

Edit Common Words

Modify the list of common words to exclude from analysis. These words are typically non-informative.

Enter the main word you want to analyze. The graph will highlight relationships with this word.

The Context Window Size defines the range of words surrounding each occurrence of the target word that are analyzed for potential relationships. This setting allows you to specify how many words before and after the target word should be included in the analysis, (this includes the context window of neighbours).

Specify how many top related words to display. This helps to focus on the most relevant connections.

0

Adjust how strongly connected words are positioned near the target word. Increasing this value brings frequently connected words closer to the target.

The Neighbors Count enhances the visualization by highlighting the most frequently co-occurring words as direct connections to each of the top frequent words in the graph. If the neighbors count is set to 3, the graph connects and adds the three most frequently co-occurring words that are next to each top word, providing clear insights into direct connections. This feature allows users to explore the strongest relationships within the dataset.

0

Adjust the influence of neighboring words. A higher weight places more connected neighbours closer to eachother.
